Panduan singkat

Cara pakai backend

1. Buka Google Sheets kosong, lalu Extensions → Apps Script.
2. Copy isi file apps-script/Code.gs ke Apps Script.
3. Simpan API key NVIDIA di Script Properties dengan nama NVIDIA_API_KEY.
4. Deploy sebagai Web App: Execute as Me, Who has access: Anyone.
5. Paste URL Web App di halaman ini, lalu klik Test koneksi.
